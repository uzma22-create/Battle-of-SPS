# 🎮 Battle of SPS

A simple Rock Paper Scissors game built in Python.

## Features
- Play against the computer
- Random computer choices
- Win, Lose, and Tie detection
- Beginner-friendly code

## Technologies Used
- Python 3
- random module

## How to Run

```bash
python Rock_Paper_Scissors.py
```
